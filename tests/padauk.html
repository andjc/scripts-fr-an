Myanmar (Padauk) font test: Firefox for Android

No typeface is specified in this page's CSS, therefore Firefox should use font fallback to select a font. Most android devices do not fonts installed that support Myanmar script.

If Myanmar text is rendered similar to what appears in the reference image, the font pack is successfully supplying a fallback font local to Firefox for Android.

Reference image

Sample text

မြန်မာဘာသာ

Expected behaviour

Before addon is installed:
Myanmar characters should display as boxes, i.e. .notdef glyph from font being used to render page.
After addon is installed:
Myanmar text should be displayed and match the reference image on test page.
After addon is disabled:
Myanmar characters should display as boxes.
After addon is re-enabled:
Myanmar characters should display as per reference image.
After addon is uninstalled:
Myanmar characters should display as boxes.
